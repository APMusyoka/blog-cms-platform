## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **CMS**: Sanity.io, Contentful, or MDX
- **Styling**: Tailwind CSS + Typography plugin
- **Newsletter**: ConvertKit or Mailchimp API
- **Comments**: Giscus or Disqus (optional)
- **Search**: Flexsearch or Algolia
- **Syntax**: Prism or Shiki
- **Deployment**: Vercel
